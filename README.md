# ✅ ICSF TOOLS Ip Tracker

![Tools](https://i.postimg.cc/hjhNpqkc/Screenshot-20250814-084809-Termux.jpg)

Welcome to **ICSF TOOLS**, the ultimate toolkit for Termux/Linux users.  
Follow the steps below to install and run.

## 🛠 **Installation:**  
**📱 Termux / Android:**  
```bash
pkg update && pkg upgrade -y  
pkg install python git -y  
git clone https://github.com/somser2icsf/ICSF_TOOLS
cd ICSF_TOOLS
chmod +x *
pip install -r requirements.txt
python3 icsf_tools-v3.py  
```  

**💻 Linux / Windows:**  
```bash
git clone https://github.com/somser2icsf/ICSF_TOOLS
cd ICSF_TOOLS
chmod +x *
pip install -r requirements.txt
python3 icsf_tools-v3.py
```
